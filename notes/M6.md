# M6 — Other Tools Tour

## Copilot — demo'd
- Observation: The text Copilot suggests while typing is much like what google docs has, and essentially predicts what you're gonna type next.

## Cursor (or Windsurf) — installed
- Observation: Unlike the others, it directly edits the code in the terminal to show you the changes, and asks you if you want to keep or undo the code.

## Gemini CLI (or Codex CLI) — installed/demo'd
- Same prompt comparison: They gave similar results, I ran this multiple times and I had a hard time deciding which was better or worse, I think they're roughly equal.

## Chat (Claude.ai or ChatGPT) — used for code review
- Observation: It pointed out some aspects of my code which may cause some issues down the line.

## Tool decision matrix
| Situation | Tool I'd reach for | Why |
| --- | --- | --- |
| Multi-file feature with good CLAUDE.md | Claude Code | Very good at reading through md files |
| 5-line helper autocomplete while typing | Copilot | This is what the feature already does |
| Brainstorm three design approaches | Chat | Use this to plan, before using something else for coding |
| Review a teammate's PR for security issues | Chat | Is good at catching errors and flaws |
| Cross-file refactor with consistent renames | Claude Code | Can sift through multiple different files at once |
| Learning a new library's API | Chat | Reviews information quickly |
